判断是否在字典里用 in 还是用 has_key()

z = { 'a':  1,  'b':  2,  'c': 3}
'a' in z
z.has_key( 'a')

官方文档推荐用 key in dict 的语法，因为它更短更通俗易懂。has_key 是老旧遗留的 API，为了支持 2.2 之前的代码留下的。Python3 已经删除了该函数。
