更新 fork 的代码

Git 上有个很方便的功能叫 fork，将别人的 Repo 一键复制到自己账号下。这个功能很方便，但有点不足的是，当源项目更新后，你 fork 的分支并不会一起更新，需要自己手动去更新。下面试更新方法：

  • clone 自己的 fork 分支到本地，命令为：
    git clone git@github.com:xxxx3/three.js.git
  • 增加源分支地址到你项目远程分支列表中(此处是关键)，命令为：
    git remote add upstream git://github.com/ssss/three.js.git
    此处可使用 git remote -v 查看远程分支列表
  • 从上游更新代码
    git pull upstream master
  • 将合并后的代码 push
    git push origin master
